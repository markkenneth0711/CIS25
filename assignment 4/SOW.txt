Project Title: Console Inventory Management System
Student Name: Mark Kenneth Miranda
Date: 07/15/2026

1. Project Overview
This C++ application is an inventory management system designed to help a user track product stock. It will utilize long-term data storage by reading from and writing to a local text file, ensuring data persists between program runs. 

2. Features
- Ability to add new product items to the inventory list
- Ability to view all existing inventory items
- Ability to save the inventory data to a local text file and load it automatically upon startup
